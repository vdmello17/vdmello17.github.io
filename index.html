Excel Dashboard Project

Online Retail Sales Performance Dashboard

Built an interactive Excel dashboard using online retail transaction data. The project involved data collection, cleaning, analysis, visualization, reporting, and business insights.

Problem

Raw transaction data was difficult to interpret because it contained product-level invoice records, returns, cancellations, and repeated invoice numbers.

Process

Cleaned the dataset, created calculated fields, separated completed sales from returns and cancellations, built Pivot Tables, and designed an interactive Excel dashboard.

Key Features

  • KPI cards for total revenue, unique orders, customers, and average order value
  • Monthly revenue trend analysis
  • Top countries by revenue excluding UK for better comparison
  • Top products and top customers by revenue
  • Returns and cancellations impact analysis
  • Interactive slicers for filtering the dashboard

Key Insights

  • The United Kingdom generated the majority of revenue.
  • November showed the highest revenue, suggesting seasonal sales behavior.
  • A small group of products and customers contributed significantly to sales.
  • Returns and cancellations had a noticeable impact and should be monitored.

Tools Used: Excel, Pivot Tables, Pivot Charts, Slicers, Data Cleaning, Dashboard Design
